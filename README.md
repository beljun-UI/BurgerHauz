# Burger Hauz
A responsive web application for a handcrafted burger shop.

## Features
- Dynamic menu page with product categories.
- Functional shopping cart using Local Storage.
- Smooth-scroll navigation and responsive design.

## How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. To view the menu, navigate to `all-burgers.html`.
